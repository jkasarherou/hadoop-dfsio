# Hadoop MR

This project aims to provide MapReduce benchmark tools for Hadoop.

It is based on the `mapreduce-examples`.


## Get started

Get the jar file and run and choose the benchmark you want:

```bash
$ yarn jar mr-*.jar
```

## Build

Using maven:

```bash
$ mvn clean package
```

## Run tests

Using mvn:

```bash
$ mvn test
```
